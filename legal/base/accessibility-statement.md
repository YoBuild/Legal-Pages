## 3. Accessibility Features

Our website includes the following accessibility features:

- Semantic HTML structure with appropriate headings and landmarks
- Text alternatives for non-text content
- Keyboard accessibility for all functionality
- Sufficient color contrast between text and backgrounds
- Resizable text without loss of functionality
- Multiple ways to navigate and find content
- Clear focus indicators for keyboard navigation
- ARIA landmarks and roles where appropriate
- Accessible forms with proper labels and error messages

## 4. Compatibility with Assistive Technologies

We aim to ensure that our website is compatible with various assistive technologies, including:

- Screen readers (such as JAWS, NVDA, VoiceOver, and TalkBack)
- Speech recognition software
- Screen magnification tools
- Alternative keyboard and mouse input devices

## 5. Known Limitations

While we strive for full accessibility, we recognize that there may be some areas of our website that are not fully accessible. We are actively working to identify and address these issues.
